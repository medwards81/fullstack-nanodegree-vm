Move Trailer Webpage Version 1.0 08/03/2015
--------------------
	This program creates a simple webpage for displaying youtube movie
	trailers and addtional movie information via the
	OMDb api (http://www.omdbapi.com).

Requirments
-----------
	The entertainment_center.py module has the following dependencies:
		*media.py
		*fresh_tomatoes.py
		
How to run
----------
	- Run the entertainment_center.py module to initiate the
	  program (e.g. via command line or Python's IDLE)
	- This module will open a webpage automagically via your default
	  browser to view the movie content
	
General Notes
-------------
	- Movie data structures are hard-coded in entertainment_center.py.
	- This could be updated to store them inside an external flat-file, 
	  or even better, to pull the movie info from a relational Db.
