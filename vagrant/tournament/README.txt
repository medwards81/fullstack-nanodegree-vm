Project 2 - Swiss Pairings Tournament
--------------------
	This package contains three files. tournament.py defines a tournament class to create and update a swiss-pairing tournament.  tournament.sql provides the postrges database schema for the tournament backend. tournament_test.py is a program used to test the methods of the tournament class.

To get started
-----------
	Run: psql \i tournament.sql on the command line to create and connect to the tournament database. Then run: python tournament_test.py on the command line to execute the test program.
